# vnz Claude Plugins

Personal Claude Code plugin marketplace by vinz.

## Installation

### From GitHub
```bash
/plugin marketplace add vnz/claude-plugins
```

### For local development
```bash
/plugin marketplace add ~/code/vnz/claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `date-context` | Injects current date/time at session start to fix web search year issues |

## Usage

After adding the marketplace:
```bash
/plugin install date-context@claude-plugins-vnz
```
